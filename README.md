# 🧬 Colorectal Cancer Risk & Survival Prediction

## 📌 Overview
This project is part of my **Data Analytics Internship**, where I analyzed colorectal cancer patient data to understand risk factors and predict patient survival using machine learning.

---

## 🎯 Objective
- Perform Exploratory Data Analysis (EDA) on healthcare data  
- Identify key factors affecting cancer survival  
- Build a predictive model to classify patient survival status  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 📂 Dataset Features
- Age, Gender, Race  
- Socioeconomic Status  
- Family History  
- Tumor Aggressiveness  
- Cancer Stage  
- Lifestyle Factors (Smoking, Diet, Alcohol)  
- Treatment Details  
- Target Variable: **Survival Status (Survived / Deceased)**  

---

## 🔍 Key Analysis
- Data cleaning and preprocessing  
- Survival distribution analysis  
- Risk factor analysis (stage, lifestyle, treatment)  
- Correlation heatmap  
- Feature encoding  

---

## 📈 Key Insights
- Majority of patients in the dataset survived  
- Dataset is imbalanced (more survived cases)  
- Lifestyle and medical factors influence survival outcomes  
- Numerical features showed weak correlation  

---

## 🤖 Model Building
- Logistic Regression model used for prediction  
- Data split into training and testing sets  
- Initial model showed bias due to class imbalance  
- Improved model using **class balancing technique**  

---

## 📊 Results
- Achieved moderate accuracy (~50%) after balancing  
- Model predicts both survival classes more fairly  
- Classification report used for detailed evaluation  

---

## 📌 Conclusion
This project demonstrates how machine learning can be applied in healthcare to analyze patient data and predict survival outcomes, while handling challenges like imbalanced datasets.
